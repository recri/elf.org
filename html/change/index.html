Change

Change is a web application for consulting the book of changes in a browser. It's self contained and can be installed as persistent web app and will run with no network connection.

The book of changes is described in detail on Wikipedia. It is a very old book which has attracted a great deal of scholarly attention over its life and accumulated a great deal of scholarly commentary in the process. This app does not provide any of this commentary, it has only the names and judgments of the 64 hexagrams, and the six or seven moving line judgments for each hexagram. This app provides the original Chinese, the Legge translation, the Wilhelm/Baynes translation, a Wilhelm/Google translation, and the Wilhelm translation in German.

This app was developed using LitElement and the PWA starter kit developed by the Polymer Project at Google. The sources for Change can be found at Github.
