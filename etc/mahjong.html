Mahjong

mahjong.elf.org is a web application for playing mahjong solitaire in a browser.

Specifically it is a progressive web application written in Polymer that plays Mahjong solitaire. As such it can be installed as a standalone web application on a computer or phone and you can play even when cut off from the net. It was developed in Chrome and I rarely try it in other browsers.
